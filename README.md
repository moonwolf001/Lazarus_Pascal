# ＜簡易説明版＞きみもPASCALでWindowsゲームを作ってみないか？

このリポジトリのブランチは、「＜簡易説明版＞きみもPASCALでWindowsゲームを作ってみないか？」に掲載されているソースコードと関連資料を含んでいます。本書はプログラムの作成後の視点で、Windowsゲーム開発のプロセスを簡潔に紹介しています。
